# AI Workflow — marketplace-infra

Как мы работаем с AI-ассистентом в этом репозитории.

## Принципы

- Перед началом задачи AI читает `infra_todo.md` и `todo-state.md` для понимания контекста
- AI предлагает план — человек подтверждает — только потом пишется код
- После завершения фазы: обновить оба файла, сохранить сессию в `docs/ai/sessions/`

## Структура документов

- `infra_todo.md` — мастер-список задач по фазам (чекбоксы)
- `todo-state.md` — живое состояние: последняя задача, следующая, принятые решения
- `docs/ai/sessions/` — архив сессий с деталями решений и итерациями
